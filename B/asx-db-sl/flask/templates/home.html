Welcome to ASX_DB


This application is an interface to a database of fundamental financial data for companies listed on the Australian Stock Exchange (ASX).

This is a fully cloud native application that has been developed using the serverless framework. It has been designed to be scalable and reliable whilst incurring minimal hosting fees.

This application demonstrates full stack web development and has been built using:

  • Serverless Framework
  • AWS Lambda
  • AWS DynamoDB (NoSQL)
  • Flask - Python Web Framework
  • Jinja2 - Python Templating Library
  • Pandas - Python Data Analysis Library

Features:

  • Severless deployment that leverages over 67 CloudFormation resources - including AWS Lambda.
  • Auto scaling DynamoDB read/write capacity.
  • Python Requirments are packaged as a seperate Lambda Layer.
  • Lambda functions are kept warm to prevent cold-starts.
  • General infomation, cash flow, balance sheet and income statement date is stored each listed company.
  • Fresh data is fetched every 2 minutes by a cron job function for the oldest ticker data in the database.
  • Threaded mutliprocessing to speed up execution time (4x) of scraping function.
  • 6 Global Secondary Indexes allows searches to be sorted by Ticker, Name, Market Cap, GICs group, Date Listed and Date Updated.
