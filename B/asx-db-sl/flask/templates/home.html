Welcome to ASX_DB


This application is an interface to a database of fundamental financial data for companies listed on the Australian Stock Exchange (ASX).

This is a fully cloud native application that has been developed using the serverless framework. It is designed to be scalable and reliable whilst incurring minimal hosting fees. This application demonstrates full stack web development capabilities using the Flask framework in combination with:
  • AWS Lambda
  • AWS DynamoDB
  • jinja
  • Pandas
